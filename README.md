## Mana Service 

匣子管理 && 元信息 && 部门信息 && 常用网站 服务

### 开发

> 首先把仓库 clone 到 $GOPATH/github.com/asynccnu 下

需要本地 Redis + Mongo，都是默认端口。Mongo admin 的账号：密码 mongoadmin:secret。

可以用 [docker 快速起一个 Mongo](https://hub.docker.com/_/mongo)。

然后在目录下 `make && ./main` 起服务
